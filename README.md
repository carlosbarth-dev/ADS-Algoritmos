# ADS - Algoritmos

Este repositório contém exercícios de algoritmos em Python, organizados em listas temáticas.

## Estrutura

- **Lista_Exercicios_python/**
  - **exercicios_repeticao/**: Exercícios sobre estruturas de repetição (loops).
  - **exercicios_selecao/**: Exercícios sobre estruturas de seleção (condicionais).
  - **Banco_dados_24-03/**: Exercícios relacionados a banco de dados (conexão, CRUD).

## Como executar

Os exercícios são scripts Python simples. Execute com `python nome_do_arquivo.py`.
